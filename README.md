## 🚀 Deployment (Netlify)

1. Push to GitHub
2. Go to [Netlify](https://netlify.com)
3. Click **"Add New Site" > "Import from Git"**
4. Connect GitHub + select this repo
5. Set build settings:
   - **Build command**: `npm run build`
   - **Publish directory**: `build`
6. Click **Deploy Site**
